About This Dashboard

This benchmarking platform was developed through a collaboration between Qualcomm Technologies and California State University San Marcos (CSUSM) to evaluate on-device performance of pre-trained semantic segmentation models available on Qualcomm AI Hub. As part of this initiative, student teams partnered with Qualcomm mentors to design and deploy tools that help developers compare model efficiency across different devices, optimizing for real-world edge AI applications. This site intends to provide transparent performance metrics to guide AI model selection for mobile and embedded systems.

What is Semantic Segmentation?

Semantic segmentation is a computer vision task that assigns a class label to each individual pixel in an image, effectively classifying every pixel into a predefined category. This process creates a dense prediction map, allowing the machine to not only identify objects but also understand their boundaries and context within the image
